# My Portfolio

Personal portfolio website showcasing my work, skills, and projects.

## Features

- About me section
- Skills & expertise
- Projects showcase
- Featured posts
- Contact information

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
